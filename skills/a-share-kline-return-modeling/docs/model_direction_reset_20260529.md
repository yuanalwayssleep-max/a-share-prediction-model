# 模型方向重置记录

## 为什么暂停继续微调

当前 compact60 主路径在 2026-03 到 2026-04 的表现仍然不稳定：

- 历史基线 Top3：`>=10%` 命中 28/129，`<=-5%` 亏损 29/129。
- 同口径当前代码重跑 Top3：`>=10%` 命中 24/129，`<=-5%` 亏损 29/129。
- 加入市场/弱市/反弹交互特征后，最好只到 `26/129`，没有超过历史基线。
- 用重要性替换回 60 特征后，命中没有稳定提升，亏损反而容易上升。

结论：继续在 compact60 内小幅换特征，已经进入低收益区。

## 粗粒度形态诊断

用 2025-10 到 2026-04 的样本，按可解释形态族群统计基础胜率。

主要发现：

- 高 `>=10%` 率的族群往往也是高亏损族群。
  - `high_pos_terminal`：`>=10%` 率 20.77%，但 `<=-5%` 率 31.32%。
  - `trend_continuation`：`>=10%` 率 14.69%，但 `<=-5%` 率 26.75%。
  - `hot_chase_risk`：`>=10%` 率 14.29%，但 `<=-5%` 率 32.59%。
- 低位、安静、弱市修复类族群亏损率较低，但 `>=10%` 率也低。
  - `quiet_low_reversal`：`>=10%` 率 4.27%，`<=-5%` 率 9.46%。
  - `low_vol_quiet_recovery`：`>=10%` 率 3.15%，`<=-5%` 率 9.02%。

这说明现有 K 线特征把“暴涨候选”和“暴跌候选”压在同一个高波动区域里，单靠当前个股 K 线字段很难稳定分开。

## 题材层验证

`00_股票清单.csv` 中已有 `概念题材`，但当前个股特征表没有每日题材热度字段。

用静态题材归属构造了每日题材热度探针：

- 题材样本：164 个题材，1479 条股票-题材关系。
- 2026-03 到 2026-04，题材热度最高分位有一定提升：
  - `max_theme_avg_turnover_rank` 最高分位：`>=10%` 率 11.69%，低分位 4.75%。
  - `max_theme_avg_range_pos_20_rank` 最高分位：`>=10%` 率 10.53%，低分位 5.63%。
  - `max_theme_ret_5_ge05_ratio_rank` 最高分位：`>=10%` 率 10.16%，低分位 6.68%。
- 但题材热度越高，亏损率也通常越高，所以题材不能单独作为选股规则。

诊断上，3-4 月高收益题材集中在 CPO、OLED、PCB、光通信、算力、数据中心、储能、充电桩等方向。这与行业字段相比，更接近 A 股短线资金实际轮动方式。

## 新方向

停止继续微调单一二分类模型，转向“题材/行业/个股三层候选生成 + 个股模型排序”。

建议只保留一套新主线：

1. 先生成每日题材热度表。
   - 从 `00_股票清单.csv` 的 `概念题材` 拆分股票-题材映射。
   - 用当日和过去收益、换手、位置构造题材热度、题材拥挤度、题材扩散度。
   - 注意只使用预测日及以前数据，不使用未来标签。

2. 个股特征表合并题材聚合特征。
   - 每只股票取其所属题材中的最大/平均题材强度。
   - 增加题材相对强度：个股强度减题材强度。
   - 增加题材拥挤风险：题材热度高 + 个股高位高换手。

3. 第一层模型不再只学“个股 K 线是否 20% 爆发”。
   - 新模型输入包含个股、行业、题材三类特征。
   - 保持固定每日 Top3，不使用不出手 gating。
   - 仍然用当前基线做验收：2026-03 到 2026-04 必须超过历史 Top3 `>=10%` 28/129，且 `<=-5%` 不超过 29/129。

## 目前不进入主路径的内容

- 市场后置 gating：不符合每日稳定 Top3。
- Top20 二层 rerank：第一层召回不足时收益有限。
- 单纯低位反转规则：胜率太低。
- 单纯高位趋势规则：暴涨率高但亏损率也高。
- 继续 compact60 内部小修小补：当前收益不足，维护成本变高。
