# 2026-03 到 2026-04 特征缺口分析

## 当前基线

- 预测窗口：2026-03-02 到 2026-04-30，共 43 个交易日。
- 当前主路径：compact60 + LightGBM + `future_5_return >= 20%` 爆发标签 + terminal risk penalty。
- Top3 合计 129 只：`>=10%` 命中 28 只，`<=-5%` 亏损 29 只。
- 3 月：66 只，`>=10%` 命中 7 只，`<=-5%` 亏损 23 只，平均收益 -1.43%。
- 4 月：63 只，`>=10%` 命中 21 只，`<=-5%` 亏损 6 只，平均收益 6.50%。

## 核心问题

第一层召回不足是主问题，不是 Top20 二层排序可以单独解决的问题。

- 全市场真实 `future_5_return >= 10%` 机会：782 只。
- 进入 Top20 候选池：105 只，占 13.4%。
- 最终 Top3 命中：28 只，占真实机会 3.6%。
- 真实 `>=10%` 机会中，3 月进入 Top20 的比例只有 10.3%，4 月是 14.8%。

Top20 仍有一定替换空间，但不是根因：

- Top20 候选池内共有 105 只 `>=10%`，最终 Top3 只拿到 28 只。
- 有 14 个交易日 Top20 里至少有 3 只 `>=10%`，但 Top3 没有选满。
- 有 12 个交易日 Top3 出现 `<=-5%` 亏损票，同时 Top20 里存在未被选上的 `>=10%` 股票。

结论：下一步应优先提升第一层对真实大涨票的识别和召回，然后再考虑候选池内排序。

## 选中亏损票的特征

和选中的 `>=10%` 赢家相比，亏损票不是典型弱票，而是更像“反弹日里的假强票”。

主要特征：

- 亏损票在市场/行业短线反弹日更容易被选中。
  - 3 月选中亏损票的 `market_avg_pct_chg` 中位数为 `+0.0584`。
  - 3 月选中赢家的 `market_avg_pct_chg` 中位数为 `-1.6580`。
  - 3 月选中亏损票的 `industry_up_ratio` 中位数为 `0.8889`。
  - 3 月选中赢家的 `industry_up_ratio` 中位数为 `0.1111`。
- 亏损票经常是前期已有涨幅、短线反弹、但后续承接失败。
  - 3 月选中亏损票 `ret_5` 中位数为 `+1.55%`。
  - 3 月选中赢家 `ret_5` 中位数为 `-2.32%`。
- 现有 terminal risk 能识别部分高位过热，但对“弱市反弹假强”识别不足。

这说明当前模型对“市场/行业当天变强”有误读：在 3 月弱市里，这类信号更像反弹陷阱，而不一定是未来 5 日强势延续。

## 漏掉的大涨票特征

模型漏掉的真实 `>=10%` 股票，整体比已选中的赢家更安静。

和选中的赢家相比，漏掉赢家的中位数特征：

- `turnover_pct`：`2.89` vs `7.88`
- `pct_chg`：`-0.19%` vs `+1.44%`
- `amp_mean_3`：`4.30` vs `6.26`
- `range_pos_60`：`0.38` vs `0.80`
- `stock_market_turnover_rank`：`0.64` vs `0.87`
- `stock_industry_ret_5_rank`：`0.71` vs `1.00`

结论：当前模型偏好高热度、高换手、高位置、行业内强排名的股票；但大量真实大涨票在起涨前并不热，甚至是低位、缩量或刚从弱势中恢复。

## 当前特征层的缺口

1. 市场状态特征使用方式不够有效。

`market_up_ratio_xrank` 这类按日期横截面 rank 的市场特征基本没有意义，因为同一天所有股票的市场值相同，横截面 rank 近似常数。真正有用的应该是原始市场状态，或者市场状态和个股形态的交互。

2. 弱市反转和低热度起涨特征不足。

当前 compact60 对热度、位置、行业内排名依赖较重，容易抓到已经热起来的票，但漏掉低位、低换手、弱转强之前的票。

3. “弱市反弹假强”没有被单独刻画。

3 月亏损票的典型模式不是极端高位追涨，而是在市场/行业短线修复时被误判为强势延续。需要把市场状态纳入个股形态判断，而不是作为后置风控。

## 建议的下一步

只做一个小闭环实验，不改主路径：

- 保持每日固定 Top3。
- 保持第一层个股模型，不做市场后置 gating。
- 在 compact60 内做一次特征收敛替换，而不是继续扩张特征数。
- 用少量新特征替换掉低价值特征：
  - 原始市场状态：`market_avg_pct_chg`、`market_up_ratio`、`market_ret_5_le_neg05_ratio`。
  - 弱市反转交互：市场弱 + 个股低位/缩量/回撤后企稳。
  - 反弹假强交互：市场或行业强反弹 + 个股短线已经涨 + 高位置/高排名。
  - 安静起涨特征：低换手但相对收益改善、低位但行业改善、近 5 日弱而 20 日不破坏。

验收标准仍然用当前基线：

- 2026-03 到 2026-04 Top3 `>=10%` 必须超过 28/129。
- `<=-5%` 亏损票不能超过 29/129。
- 重点看 3 月是否从 7/66 明显改善，不能只靠 4 月上涨行情抬分。
