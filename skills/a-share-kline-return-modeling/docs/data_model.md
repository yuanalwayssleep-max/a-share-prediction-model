# 股票预测数据模型与清洗逻辑

## 目标

先把 `data/` 下的原始数据整理成稳定、可复跑、可审计的数据层。后续模型不直接读取原始 CSV，而是读取清洗后的标准表。

## 数据分层

### 原始层 Raw

只读，不改写：

- `data/单只股票日k/*.csv`：一只股票一个日K文件。
- `data/指数日K文件/00_核心指数日K.csv`：核心指数日K汇总。
- `data/a股快照历史/*.csv` 和 `data/a股快照_*.csv`：股票元数据和行业/板块快照。

### 清洗层 Clean

输出目录：

- `skills/a-share-kline-return-modeling/data/`

核心表只保留两张：

- `个股k线特征数据.csv`：清洗后的股票日K + 元数据 + 质量标记 + 特征工程 + 未来5日训练标签。后续单股预测模型只读取这一张表。
- `指数k线特征数据.csv`：清洗后的核心指数日K + 指数基础特征 + 未来5日训练标签。后续市场预测模型读取这一张表，股票表中也会合并常用指数特征。

这两张表是模型唯一数据入口。`future_5_*` 字段允许落表，但它们是标签列，不是特征列。任何训练、回测、预测脚本都必须把 `future_*` 从模型输入 `X` 中排除，只能作为训练/回测答案 `y` 使用。

## 股票日K标准字段

`个股k线特征数据.csv` 字段：

- `trade_date`：交易日期。
- `symbol`：6位股票代码。
- `name`：股票名称。
- `exchange`：交易所，推断或来自快照。
- `board`：板块，优先来自快照。
- `industry`：行业，优先来自快照。
- `open`、`close`、`high`、`low`：开收高低价。
- `volume`、`amount`：成交量、成交额。
- `amplitude_pct`：振幅，百分点。
- `pct_chg`：涨跌幅，百分点。
- `chg_amount`：涨跌额。
- `turnover_pct`：换手率，百分点。
- `source_file`：来源文件。
- `limit_pct`：该股票理论涨跌停幅度。
- `is_st`：是否 ST。
- `near_limit_up`、`near_limit_down`：是否接近涨跌停。
- `natural_day_gap`：距上一条记录的自然日间隔。
- `future_5_trade_date`、`future_5_close`、`future_5_return`、`future_5_direction`、`future_5_up_label`：未来5个交易日标签，只能作为训练/回测答案。
- `is_training_eligible`：是否建议纳入训练。
- `ret_1/3/5/10/20`、均线乖离、量能比、波动率、K线形态、20日区间位置：个股特征。
- `is_pre_holiday_3`、`is_post_holiday_3`、`pre_holiday_tday`、`post_holiday_tday`、`holiday_gap_days`：由交易日长休市间隔自动生成的节假日前后特征。
- `market_*`：当日股票池市场宽度和整体涨跌特征。
- `industry_*`：行业宽度和涨跌特征。
- `sh_*`、`hs300_*`、`zz500_*`、`zz1000_*`：核心指数特征。

## 清洗规则

### 硬异常

硬异常会从 `个股k线特征数据.csv` 中剔除：

- 日期或代码无效。
- `日期 + 代码` 重复。
- 开盘、收盘、最高、最低、成交量、成交额缺失。
- 价格小于等于 0。
- 最高价低于开/收/低任一价格，或最低价高于开/收/高任一价格。
- 成交量或成交额小于等于 0。

### 软异常

软异常保留在 `个股k线特征数据.csv`，但 `is_training_eligible=0`：

- 接近涨停或跌停。
- 涨跌幅超过理论涨跌停幅度。
- 振幅大于 40%。
- 换手率大于 80%。
- 涨跌幅与前收盘复核差异较大。
- 特征生成所需字段缺失。

### 训练过滤原则

训练模型时读取 `个股k线特征数据.csv`，并使用 `is_training_eligible == 1` 过滤历史训练样本。

预测锚点日股票池不应直接使用训练过滤结果删除股票。训练过滤只用于历史训练样本，避免极端异常污染模型。

未来5日上涨/下跌标签可以落在这两张表中，但必须按以下规则使用：

- `future_*` 只能作为 `y`。
- `future_*` 永远不能进入模型特征 `X`。
- 真实预测锚点日时，即使 `future_*` 为空，也必须按同一套禁止特征清单过滤。
